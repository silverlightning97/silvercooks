# SilverCooks CMS-ready website

This version is meant for:

- GitHub repository storage
- Cloudflare Pages hosting
- Pages CMS editing

## What you edit later

Use Pages CMS to edit:

- Recipes
- Restaurant Memories
- Images

Do not manually upload ZIP files to Cloudflare after this setup. Push/edit files in GitHub, or use Pages CMS. Cloudflare will update automatically when the GitHub repository changes.

## Files

- index.html: website page
- style.css: design
- app.js: loads the recipes/restaurants
- data/recipes.json: recipes edited by Pages CMS
- data/restaurants.json: restaurant memories edited by Pages CMS
- .pages.yml: Pages CMS admin configuration
- images/: uploaded pictures will go here
